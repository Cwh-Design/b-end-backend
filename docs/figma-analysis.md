## 2. 组件清单

**结论：该设计稿未使用 Figma 组件化系统。**

- COMPONENT 数量：**0**
- COMPONENT_SET 数量：**0**

所有重复元素（导航栏、卡片、表格行）均以 **GROUP（编组）+ RECTANGLE / TEXT** 的方式复制实现，而非 COMPONENT Instance。

> ⚠️ 这意味着无法直接从 JSON 提取组件属性（variants/properties），需要人工逐帧分析视觉规范。

### 按功能分类的重复模块（非组件实例）：

| 模块 | 出现页面 | 结构 |
|------|---------|------|
| 左侧导航栏 | 所有页面 | GROUP，内含 Rectangle + 菜单项 + 常用应用 |
| 顶部搜索栏 | 所有页面 | 搜索框 + 通知图标 + 用户头像 |

## 8. 缺失状态

以下在前端还原时需要注意补充的状态，设计稿中未体现：

### 8.1 组件交互状态

| 状态 | 说明 | 建议 |
|------|------|------|
| **Hover** | 所有按钮、菜单项、表格行为悬浮态 | 加深背景 5-10% |
| **Pressed/Active** | 按钮按下态 | 加深背景 10-15% |
| **Focused** | 输入框、按钮聚焦态 | 使用 `--ring` 蓝色外发光 |
| **Disabled** | 按钮/输入框禁用态 | 降低不透明度至 40-50%，灰色文字 |
| **Loading** | 数据加载中 | 骨架屏或 Spinner |
| **Empty** | 列表/表格无数据 | 空状态插画 + 引导文案 |

### 8.2 全局组件

| 组件 | 说明 |
|------|------|
| **弹窗 (Modal/Dialog)** | 新增/编辑/确认删除等操作需要弹窗 |
| **Toast/通知** | 操作成功/失败反馈 |
| **下拉选择器** | 表单中的下拉选项 |
| **日期选择器** | 筛选/表单中的日期选择 |
| **Tooltip** | 图标按钮的文字提示 |
| **分页器** | 表格底部分页控件 |
| **面包屑** | 深层页面的导航路径 |

### 8.3 响应式/边界

| 状态 | 说明 |
|------|------|
| 小屏适配 | 当前固定 1920px，需定义 < 1440px 的侧边栏收折/隐藏逻辑 |
| 长文本截断 | 表格列宽固定时，长文本的省略规则 |
| 数据溢出 | 数值过大时的格式化（如 9999+） |

### 8.4 权限/角色

| 状态 | 说明 |
|------|------|
| 无权限页面 | 403 禁止访问 |
| 菜单权限控制 | 不同角色可见不同菜单 |
| 操作按钮权限 | 编辑/删除按钮按角色显隐 |

## 9. 建议补充的 Auto Layout

以下模块当前使用绝对定位/手动编组，建议转换为 Auto Layout 以便前端还原：

### 9.1 高优先级

| 模块 | 当前方式 | 建议 | 理由 |
|------|---------|------|------|
| **左侧导航菜单** | 手动 GROUP | VERTICAL, itemSpacing: 34 | 菜单项需动态增删 |
| **顶部导航栏** | HORIZONTAL (已有) | ✅ 保持，但建议标注固定高度 80px | — |
| **统计卡片组** | SPACE_BETWEEN | ✅ 保持，补充卡片最小宽度和换行规则 | 响应式需要 |
| **数据表格** | 手动 RECTANGLE | VERTICAL 容器 + 行组件 | 表格需支持排序/筛选/分页 |
| **表单区域** | VERTICAL, itemSpacing: 20 | ✅ 保持，建议加 label 与 input 的间距规范 | — |
| **待办列表** | VERTICAL, itemSpacing: 22 | ✅ 保持 | — |

### 9.2 中优先级

| 模块 | 建议 |
|------|------|
| **LOGO 区域** | HORIZONTAL, itemSpacing: 12, align: CENTER |
| **搜索框** | HORIZONTAL, icon + input, 固定高度 43px |
| **用户信息区** | HORIZONTAL, avatar + name, itemSpacing: 10 |
| **应用收藏卡片** | HORIZONTAL, itemSpacing: 40, 超过容器宽度时换行 |
| **流程节点** | VERTICAL, itemSpacing: 20, 各节点统一圆角 |
| **图表图例** | HORIZONTAL, itemSpacing: 18 |

### 9.3 低优先级

| 模块 | 建议 |
|------|------|
| 常用应用列表 | VERTICAL, itemSpacing: 18 |
| 标签组 | HORIZONTAL, itemSpacing: 8 或 12 |
| 流程属性组 | VERTICAL + HORIZONTAL 嵌套，itemSpacing: 22 / 32 |
